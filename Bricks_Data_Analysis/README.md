# Bricks AI Data Analysis Project

This project was created using the **Bricks AI Data Analysis tool**.
The goal of this project is to analyze data and present insights using
interactive analytical bricks and visual dashboards.

## Project Overview
- Data was analyzed using Bricks AI
- Multiple analytical bricks were created
- Insights are presented through visual dashboards
- Screenshots are included to demonstrate the analysis results

## Key Features
- Automated data analysis using AI
- Clear visualization of trends and metrics
- User-friendly dashboard layout
- Insight-driven bricks for decision making
